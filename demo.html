To demonstrate the benefits of the timeout script, you must first observe the behavior of the setInterval / setTimeout functions when the tab is not active.
To do so, two timers will be created to modify the tab title:

  • I: the number of 1/100 seconds counted using setInterval
  • T: the number of 1/100 seconds counted using setTimeout
Click to run the timers.
Then open another tab and see how this tab title evolves.
